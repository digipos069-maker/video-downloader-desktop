
# Social download manager

A cross-platform desktop application for downloading videos and photos from various platforms.

## Features (Planned)

- **Multi-Platform Support:** Download from YouTube, Instagram, Pinterest, TikTok, and Facebook.
- **Rich Download Control:** Pause, resume, and cancel downloads.
- **Concurrent Downloads:** Download multiple files at once.
- **Queue Management:** See all your planned downloads in one place.
- **Format Selection:** Choose video resolution and format.
- **Desktop Notifications:** Get notified when downloads complete or fail.

## Disclaimer

This application is for personal and educational use only. Downloading copyrighted content without permission may be illegal in your country. The developers of this application are not responsible for its misuse. Please respect the terms of service of the websites you download from.

## Usage

1.  **Set up the environment:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: .\venv\Scripts\activate
    pip install -r requirements.txt
    ```
2.  **Run the application:**
    ```bash
    python main.py
    ```

## Known Limitations

- This is a work in progress. Many features are not yet implemented.
- Platform handlers are currently stubs and do not perform real downloads.
- Error handling is minimal.
